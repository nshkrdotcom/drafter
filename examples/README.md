# Drafter Examples

Run any example with:

```bash
elixir examples/<name>.exs
```

| Example | Description |
|---------|-------------|
| `hello_world.exs` | Minimal app — a good starting point |
| `counter.exs` | Stateful counter with increment/decrement buttons |
| `widgets.exs` | Showcase of common widgets: buttons, text input, checkbox, switch, option list, progress bar |
| `screens.exs` | Modal dialogs and toast notifications |
| `calculator.exs` | Calculator with keyboard and mouse input |
| `charts.exs` | Line, bar, area, candlestick, and scatter charts with live simulation |
| `data_table.exs` | Sortable, scrollable DataTable with 100 rows |
| `clock.exs` | Live clock using the Digits widget |
| `animation.exs` | Color animation with static, pulse, and rainbow modes |
| `themes.exs` | Theme switcher using Switch widgets |
| `hsl_colors.exs` | HSL, RGB, and hex color format demo |
| `key_inspector.exs` | Live view of all keyboard, mouse, and resize events — useful when building keybindings |
| `theme_sandbox.exs` | Full widget gallery with live theme switching |
| `custom_loop.exs` | Advanced: bring-your-own event loop using low-level APIs |
